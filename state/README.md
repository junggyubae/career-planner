# state/ — who you are (past + present)

Your career **state** `s` — the facts the card reasons over.

- `identity.md` — name, contact, links, current status
- `experience/` — one file per item (frontmatter-dated); `TIMELINE.md` is **derived**
- `interests.md` — research / career interests
- `beliefs.md` — **your core identity**: values + how you act on them (the anchor the card leans on hardest)
- `uploads/` — raw CVs / SOPs / templates you feed in

Private (git-ignored); only READMEs are tracked.
