# uploads/

Raw files you upload — CVs, SOPs, transcripts, and **your own CV/SOP LaTeX templates** if you have them.

- Info Retrieval saves uploaded documents here, then *refines* their facts into the other memory folders.
- Alignment reads your template from here if present (e.g. `cv-template.tex`, `sop-template.tex`); otherwise it falls back to the bundled default in `card/templates/`.

This folder is **private** (git-ignored). Only this README is committed.
